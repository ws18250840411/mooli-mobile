# Image 图片

### 介绍

增强版的 img 标签，提供多种图片填充模式，支持图片懒加载、加载中提示、加载失败提示。

## 代码演示

### 基础用法

基础用法与原生 `img` 标签一致，可以设置 `src`、`width`、`height`、`alt` 等原生属性。

```jsx
import { Button } from 'mooli-mobile';

const Demo = () => {
  return <div className="demo-name"></div>;
};

ReactDOM.render(<Demo />, mountNode);
```
